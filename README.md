# Binance USDT-M 선물 트레이딩 계산기

Binance USDT-M 선물 거래를 위한 포지션 사이즈 및 리스크 관리 계산기입니다.

## 기능

- **리스크 기반 포지션 사이즈 계산**: 총 자산 대비 리스크 비율을 정확히 만족하는 포지션 크기 산출
- **수수료 포함 계산**: Taker 수수료(0.04% 왕복)를 포함한 실제 손익 계산
- **레버리지 최적화**: 5~20배 범위 내에서 최적 레버리지 계산
- **손익비 분석**: 이론적 및 실제 손익비(R/R) 계산
- **구조적 문제 검증**: 진입가, 손절가, 익절가 간의 논리적 관계 검증
- **종합 판정**: 포지션의 안전성 및 조정 필요 여부 판정

## 설치

```bash
pip install -r requirements.txt
```

## 실행

```bash
streamlit run app.py
```

## 사용 방법

1. **총 자산**: 거래에 사용할 총 자산 입력 (USD)
2. **최대 리스크 비율**: 총 자산 대비 리스크 비율 설정 (5~20%)
3. **포지션 방향**: LONG 또는 SHORT 선택
4. **가격 정보**: 진입가, 스탑로스, 1차/2차 익절가 입력
5. **계산하기** 버튼 클릭

## 계산 결과

계산 결과는 다음 정보를 포함합니다:

1. **손절 정보**: 손절 폭(%), 손절가, 실제 손실 금액
2. **포지션 크기**: 적정 Notional 및 수량
3. **레버리지**: 포지션 사용 레버리지 및 실질 레버리지
4. **손익비**: 1차/2차 손익비 및 실제 손익비 기반 순이익
5. **구조적 문제**: 가격 구조상의 문제 여부
6. **필요 Margin**: 해당 포지션에 필요한 Margin
7. **종합 판정**: 문제 없음 / 사이즈 조정 필요 / SL 조정 필요
8. **실제 진입 포지션**: 최종 진입 정보

## Alert 메시지

계산 후 생성되는 Alert 메시지를 복사하여 거래 알림에 사용할 수 있습니다.

## 주의사항

- Binance USDT-M 선물 기준으로 계산됩니다
- Isolated Margin 모드를 가정합니다
- Taker 수수료(0.04% 왕복)가 포함됩니다
- 실제 거래 시 시장 변동성 및 슬리피지를 고려하세요
